To use, 
run ```python app.py``` in project directory
open http://127.0.0.1:5000

To change run conditions, modify lines 33 or 34
33 refers to novelty_seeking scale factor, so a value > 1 means people want to try new places, < 1 means they don't
34 refers to goal, which can be maximizing minimum utility, maximizing total utility, or maximizing nash welfare
